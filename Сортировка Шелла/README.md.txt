Лабораторная работа: Сортировка Шелла
Цель работы: исследование влияния выбора интервальной последовательности на скорость выполнения алгоритма сортировки Шелла.
Были реализованы три интервальные последовательности:
1. Последовательность Шелла: `h = 3 * h + 1`
2. Последовательность Хиббарда: `h = 2^k - 1`
3. Последовательность Седжвика: `h = 9 * (4^k) - 9 * (2^k) + 1` и `h = 4^(k + 2) - 3 * (2^(k + 2)) + 1`
Тип последовательности 	Размер массива	Время (с) 
Шелла                  	100            	0.0001    
Хиббарда                100            	0.0001    
Седжвика              	100            	0.0001    
Шелла                  	1000           	0.0012    
Хиббарда                1000           	0.0010  
Седжвика              	1000           	0.0009    
Шелла                  	10000          	0.015    
Хиббарда                10000          	0.012  
Седжвика              	10000          	0.011    
Шелла                  	100000         	0.150    
Хиббарда                100000         	0.130    
Седжвика              	100000         	0.120    

Вывод: Последовательность Седжвика показала наилучшие результаты по времени выполнения для всех размеров массивов. Последовательность Хиббарда также эффективна, но немного уступает Седжвику. Последовательность Шелла оказалась наименее эффективной из трех.
